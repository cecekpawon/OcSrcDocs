IN UINT32  LoadedHeaderSize,
OUT VOID *  Destination,
IN UINT32  DestinationSize 
)

Loads the Image Sections into the memory space and initialises any padding with zeros.

Parameters
[in]ContextThe context describing the Image. Must have been initialised by PeCoffInitializeContext().
[in]LoadedHeaderSizeThe size, in bytes, of the loaded Image Headers.
[out]DestinationThe Image destination memory.
[in]DestinationSizeThe size, in bytes, of Destination. Must be at least Context->SizeOfImage + Context->SizeOfImageDebugAdd. If the Section Alignment exceeds 4 KB, must be at least Context->SizeOfImage + Context->SizeOfImageDebugAdd Context->SectionAlignment.

Load the Image into the destination memory space.

Parameters
[in]ContextThe context describing the Image. Must have been initialised by PeCoffInitializeContext().
[out]DestinationThe Image destination memory. Must be allocated from page memory.
[in]DestinationSizeThe size, in bytes, of Destination. Must be at least Context->SizeOfImage + Context->SizeOfImageDebugAdd. If the Section Alignment exceeds 4 KB, must be at least Context->SizeOfImage + Context->SizeOfImageDebugAdd Context->SectionAlignment.
Return values
RETURN_SUCCESSThe Image was loaded successfully.
otherThe Image could not be loaded successfully.
