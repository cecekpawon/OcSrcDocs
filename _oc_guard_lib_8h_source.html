62//
63// The interfaces below provide base safe arithmetics, reporting
64// signed integer overflow and unsigned integer wraparound similarly to
65// os/overflow.h in macOS SDK.
66//
67// Each interface may be implemented not only as an actual function, but
68// a macro as well. Macro implementations are allowed to evaluate the
69// expressions no more than once, and are supposed to provide faster
70// compiler builtins if available.
71//
72// Each interface returns FALSE when the the stored result is equal to
73// the infinite precision result, otherwise TRUE. The operands should
74// be read left to right with the last argument representing a non-NULL
75// pointer to the resulting value of the same type.
76//
77// More information could be found in Clang Extensions documentation:
78// http://releases.llvm.org/7.0.0/tools/clang/docs/LanguageExtensions.html#checked-arithmetic-builtins
79//
