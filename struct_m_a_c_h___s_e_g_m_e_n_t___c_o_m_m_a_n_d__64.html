Detailed Description

The 64-bit segment load command indicates that a part of this file is to be mapped into a 64-bit task's address Sectionace. If the 64-bit segment has sections then section_64 structures directly follow the 64-bit segment command and their size is reflected in cmdsize.

for 64-bit architectures
