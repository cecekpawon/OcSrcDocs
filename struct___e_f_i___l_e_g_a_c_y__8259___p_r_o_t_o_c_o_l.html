Detailed Description

Protocol Description:
Abstracts the 8259 and APIC hardware control between EFI usage and Compatibility16 usage.
Parameters
SetVectorBaseSets the vector bases for master and slave PICs.
GetMaskGets IRQ and edge/level masks for 16-bit real mode and 32-bit protected mode.
SetMaskSets the IRQ and edge\level masks for 16-bit real mode and 32-bit protected mode.
SetModeSets PIC mode to 16-bit real mode or 32-bit protected mode.
GetVectorGets the base vector assigned to an IRQ.
EnableIrqEnables an IRQ.
DisableIrqDisables an IRQ.
GetInterruptLineGets an IRQ that is assigned to a PCI device.
EndOfInterruptIssues the end of interrupt command.
