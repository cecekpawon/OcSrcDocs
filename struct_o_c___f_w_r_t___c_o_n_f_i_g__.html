◆ WriteUnprotector

BOOLEAN OC_FWRT_CONFIG_::WriteUnprotector

Make UEFI runtime services drop CR0 WP bit on calls to allow writing to read only memory. This workarounds a bug in many types of APTIO firmware that do not survive W^X. Latest Windows brings Virtualization-based security and monitors CR0 by launching itself under a hypevisor. Since we need WP disable on macOS to let NVRAM work, and for the time being no other OS requires it, here we decide to use it for macOS exclusively.
